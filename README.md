# Cloud code Studio

## Cloud Code Studio

An online code editor like VISC, built with React +typescript +Vite +Tailwind CSS
.

## Features

- File tree sidebar for exploring project files
- code editor with syntax highlighting
- command palette for quick access to commands - \u0037cs 'Command Palette'
X- Tab bar for multiple files
- Terminal emulator for running commands
- Status bar for displaying file information
- Lighthmem and dark mode support

## Technologies

- Frontend: React + TypeScript
- Build: Vite
CSS: Tailwind CSS
State Management: Zitandstore
